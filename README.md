# 个人博客网站

这是一个从零开始搭建的静态个人博客。当前版本不需要安装依赖，可以直接用浏览器打开 `index.html`，也可以用本地预览服务器查看。

## 文件说明

- `index.html`：博客页面结构，包括首页、文章列表、关于我和订阅区域。
- `styles.css`：页面样式和响应式布局。
- `script.js`：文章数据、分类筛选和订阅表单演示逻辑。
- `assets/hero-blog.png`：博客首页封面图。
- `server.mjs`：本地预览服务器。

## 如何预览

```bash
npm run dev
```

启动后打开 `http://localhost:4173`。

## 如何修改个人信息

1. 打开 `index.html`。
2. 把 `我的博客`、首页大标题、简介和“关于我”部分改成你的内容。
3. 如果你有自己的头像、照片或封面图，可以放到 `assets/` 目录，再修改 HTML 里的图片路径。

## 如何新增文章

现在文章数据写在 `script.js` 顶部的 `posts` 数组里。复制其中一段对象，然后改标题、日期、分类和摘要：

```js
{
  title: "新文章标题",
  date: "2026-06-08",
  category: "写作",
  excerpt: "这里写文章摘要。",
}
```

下一步可以继续升级为独立文章页、Markdown 写作、标签页、搜索、RSS 和部署到 GitHub Pages 或 Vercel。
